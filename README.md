## 🚀 Sobre o Projeto
O HardDex visa simplificar a busca por componentes de computador, traduzindo especificações técnicas complexas para uma linguagem acessível ao usuário comum, através de:
- **Busca Semântica:** Filtros inteligentes baseados em linguagem natural.
- **Perfilamento (TDU):** Recomendações personalizadas de acordo com o Tipo de Uso.
- **Dicionário Contextual:** Explicações simples para termos técnicos.

### 🛠 Tecnologias Envolvidas
- **Frontend:** Vue.js e Nuxt.js.
- **Backend:** Node.js e Express.
- **Banco de Dados:** SQLite com Sequelize ORM.
